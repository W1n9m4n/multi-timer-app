# Multi Timer & Schachuhr PWA

GitHub-Pages-kompatible iPad/iPhone-Web-App.

## Funktionen

- Mehrere programmierbare Timer
- Timer-Boards speichern/laden/duplizieren/löschen
- Schachuhr-Dashboard
- Aktivzeit und Zugzähler pro Spieler
- Schachuhr-Alarm endet automatisch nach 10 Sekunden
- Bearbeiten-Modus für iPad statt Long-Press-Zwang
- Offline-Cache nach erstem Laden

## Wichtig für iPad/iPhone

Für Timer-Einstellungen:
1. Im Timer-Tab oben auf `Bearb.` tippen.
2. Danach Timer antippen.
3. Einstellungen ändern.
4. Wieder `Fertig` tippen.

Long-Press bleibt zusätzlich vorhanden, ist auf iPadOS aber nicht immer zuverlässig.

Für Schachuhr-Einstellungen:
- Im Schachuhr-Tab oben rechts auf das Zahnrad tippen.
